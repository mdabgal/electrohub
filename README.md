##  Installation

Clone the repository

```bash
git clone https://github.com/mdabgal/electrohub.git
```

Go to project folder

```bash
cd electrohub
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Build for production

```bash
npm run build
```
